## About

This is a fork of Auxio where I added some features for my personal use, mostly per-word lyrics and 2 stems audio to be able to do some karaoke.

Forgive the code quality, it's partly (mostly) vibe coded.


## Additonal Features

- Added lyrics support (*.ttml format)
- Added support for 2 stems audio (the main audio file has its neighboring *\*_vocals.\** and *\*_accompaniment.\** files, users change the volume of each stem, or mute them entirely).
